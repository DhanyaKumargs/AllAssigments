What is website?

A website is a collection of related web pages, including multimedia content, typically identified with a common domain name, and published on at least one web server.

Web browser

A web browser (commonly reffered to as a browser) is a software application for accessing information on the world wide wed. Each individual web page, image and video is identified by a distinct URL, enabling browser to retrieve and display them on the user,s device.
